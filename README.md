# WABAGS E-commerce ??

**WABAGS** es una tienda virtual dedicada a la comercialización de productos artesanales Wayuu, creada como parte de un laboratorio académico de desarrollo de e-commerce.

Este proyecto integra conocimientos en **HTML, CSS, JavaScript y control de versiones con Git y GitHub**, ofreciendo una experiencia de compra funcional, visualmente atractiva y centrada en la tradición cultural Wayuu.

## ?? Características principales

- Catálogo de productos con imágenes y precios
- Carrito de compras funcional con almacenamiento local
- Modal de vista previa para cada producto
- Buscador de productos en tiempo real
- Integración con WhatsApp para realizar pedidos
- Responsive design (adaptable a dispositivos móviles)
- Publicación del sitio con GitHub Pages

## ?? Tecnologías utilizadas

- HTML5
- CSS3 (con estilo moderno e interactivo)
- JavaScript Vanilla (JS puro)
- Git y GitHub para control de versiones
